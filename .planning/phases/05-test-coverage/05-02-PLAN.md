isColdOutreach() — negative cases:
- 'Hey, want to grab coffee?' → false (no job keywords)
- 'Meeting rescheduled to Friday' → false
- 'Your order has been shipped' → false

isColdOutreach() — edge cases:
- HTML-only message body: '&lt;p&gt;We are &lt;strong&gt;hiring&lt;/strong&gt; interns&lt;/p&gt;' → the classifier normalizes text; since html tags become spaces, 'hiring' and 'interns' tokens survive → true
- Forwarded prefix: 'Fwd: [Stripe] Internship opportunity' → true ('internship' keyword matches)
- Fuzzy match: 'internshipp' (typo with edit distance 1 from 'internship' which is 10 chars → limit 2, so matches) → true
- Non-ASCII body: 'Wir suchen Praktikanten' (German, no English keywords) → false (no keyword match)

countKeywordMatches() — counts:
- 'intern hiring' → 2 (two distinct groups matched)
- 'internship internships intern' → 1 (same group, counts once)
- 'resume cv application' → 2 (resume group + apply group)
- 'no job keywords here' → 0

extractCompanyFromEmail() — tests:
- 'rachel@rocketbrew.com' → 'Rocketbrew'
- 'hr@stripe.com' → 'Stripe'
- 'noreply@google.com' → 'Google'
